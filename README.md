# InsightFocus


## 本地部署
### 拉取代码到本地
> ```shell
> git clone https://github.com/rv192/InsightFocus.git  && cd InsightFocus
> ```
### 配置venv环境
> ```shell
> python3 -m venv venv && source venv/bin/activate
> ```
### 安装项目依赖
> ```shell
> pip install -r requirements.txt
> ```
### 运行
> ```shell
> python main.py
> ```
